# ReSpace AI Docs

This folder organizes the product and technical handoff documents for the Flutter Phase 1 MVP.

- `ai_room_reshuffle_redesign_prd.pdf`: product requirements.
- `ai_room_reshuffle_redesign_trd.pdf`: technical requirements and future backend architecture.
- `ai_room_reshuffle_redesign_app_flow.pdf`: end-to-end product flow.
- `design-handoff-notes.md`: implementation notes distilled from the Claude Design export.

The original Claude Design export remains unchanged in `design-handoff/AI Design/`.
